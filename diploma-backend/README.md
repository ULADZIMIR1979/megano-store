# Интернет-магазин MegaNo

## Описание проекта

Проект представляет собой интернет-магазин с использованием Django REST Framework в качестве backend и готового frontend решения в виде пакета diploma-frontend.

## Архитектура

Проект состоит из следующих основных компонентов:

- Backend (дипломный проект): основная логика приложения, реализованная на Django
- Frontend (diploma-frontend): готовое решение для веб-интерфейса, подключаемое как пакет

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone <url>
   cd python_django_diploma
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

3. Установите пакет frontend:
   ```
   pip install diploma-frontend/dist/diploma_frontend-0.6.tar.gz
   ```

4. Выполните миграции:
   ```
   python3 manage.py migrate
   ```

5. Загрузите фикстуры:
   ```
   python manage.py loaddata shop/fixtures/demo_data.json
   python manage.py loaddata users/fixtures/users.json
   python manage.py loaddata orders/fixtures/orders.json
   ```

6. Запустите сервер:
   ```
   python3 manage.py runserver
   ```

## Структура проекта

- `shop` - приложение для работы с товарами, категориями, отзывами
- `orders` - приложение для работы с заказами, корзиной, платежами
- `users` - приложение для работы с пользователями

## API

Документация к API доступна по адресу: http://127.0.0.1:8000/swagger/

## Админ-панель

Доступ к админ-панели: http://127.0.0.1:8000/admin/

## Особенности реализации

- Все модели и представления соответствуют требованиям из swagger.yaml
- Реализованы все необходимые API-эндпоинты для работы интернет-магазина
- Интеграция с frontend решением через API
- Использование Django REST Framework для создания RESTful API
- Реализация аутентификации и авторизации пользователей
- Поддержка работы с корзиной и заказами
- Реализация системы отзывов и рейтинга товаров
- Интеграция с системой оплаты
